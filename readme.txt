Описание проекта
Это страница новостного сайта с адаптивным дизайном. Она включает:

Шапку (header): меню навигации, кнопки входа/регистрации, переключатель темной темы.
Основной блок (main): раздел с последними новостями, слайдер, фильтры для сортировки постов.
Футер (footer): копирайт и ссылки на соцсети.
Использованные технологии
HTML5: Основная разметка.
CSS: Пользовательские стили (style.css, darkmode-toggle.css, mediaqueries.css).
Bootstrap: Компоненты и сетка для адаптивности.
JavaScript: Логика для переключения темной темы и фильтров постов.
Структура страницы
Шапка (Header):

Адаптивное меню (Bootstrap).
Кнопки входа/регистрации.
Переключатель темы с чекбоксом.
Основной блок (Main):

Recent News: карточки новостей.
Слайдер: баннеры с автопрокруткой (Bootstrap).
Фильтры: кнопки для сортировки по категориям.
Футер (Footer):

Копирайт.
Ссылки на соцсети.
Подключенные скрипты
app.js: Управляет сменой темы и фильтрацией постов.
script.js: Дополнительные функции, например, переключение кнопок входа/выхода.
Основные шаги разработки
Создана HTML-разметка с Bootstrap-компонентами.
Добавлены стили для кастомизации.
Написаны скрипты для интерактивности (тема, фильтры).
Проведено тестирование на разных устройствах.
